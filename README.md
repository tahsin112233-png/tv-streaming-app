# 📺 TV Streaming App

Watch live TV anywhere with 4G/5G data! No WiFi needed.

## 🎯 How It Works

1. Open the app
2. Select **TV Server 1-8**
3. Choose a **Channel**
4. Tap **▶️ Play Stream**
5. Watch live! 🎬

## ✨ Features

✅ 8 Live TV Servers  
✅ 10+ Channels Each  
✅ Works on Mobile Data  
✅ One-Click Streaming  
✅ No Login Required  
✅ Clean Interface  

## 🚀 Quick Deploy

1. Copy this repo
2. Go to [vercel.com](https://vercel.com)
3. Import from GitHub
4. Click Deploy
5. Get live URL in 2 min!
